### Decisions & Assumptions Made
- **Killed the floating pill nav** — `border-radius: 3rem` + centered floating header is extremely common across portfolios in 2024-25. The flush full-width strip reads as more deliberate and less templated.
- **Split-color TM monogram** — `T` in `--accent-color` (purple), `M` in near-white. Creates a real brand mark feel using only existing fonts (Bricolage Grotesque 800, already loading). Dropped the logo image and Barlow Condensed name span entirely.
- **Live dot is always green** — not purple, not themed. Green = "online / active / live signal" — universally readable. Connects to data monitoring and EDM stage-light aesthetics.
- **Active indicator is a dot at top of strip, not an underline** — underline growing from 0→100% is the single most overused nav pattern. Small 4px dot at `top: 6px` reads like a control panel LED.
- **Mobile overlay is always dark** — even when the site is in light mode. The full-screen dark menu creates dramatic contrast and avoids the awkwardness of a jarring white overlay.
- **Asymmetric hamburger bars** — `22px / 14px / 22px`. The shorter middle bar is a subtle personality detail; signals the design was made, not generated.
- **Bottom nav removed completely** — user explicitly requested no bottom nav. Staggered overlay replaces it entirely, works on all mobile sizes.
- **Assumed 11ty build pipeline** — source is in `src/`, CSS/JS are in root-level `assets/`. Edits to `src/_layouts/base.njk` rebuild via `npm run build`.
